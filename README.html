sortUml() - sort german vowel mutations naturally

The Script changes the vowel mutaion's ASCCI code to halfsteps of their original vowel and sorts the array based on that fake ASCII codes

Usage:

var arr 	= ['abc', 'äbc', 'Äbc', 'Abc'],
	sorted	= arr.sort( sortUml );
console.log( sorted );

view output (or try for yourself in the console)

["Abc", "Äbc", "abc", "äbc"]
